# EcoCycle – Recycling Education Website

EcoCycle is a simple educational web application designed to help people understand how to recycle materials properly. The website provides information on different types of recyclable materials, explains how they should be prepared before recycling, helps users locate nearby recycling centers, and includes an interactive quiz to test recycling knowledge.

The project focuses on promoting environmental awareness through a clean, user-friendly interface and interactive components built with basic web technologies.

**Live Website:**  
https://etor01.github.io/HCI/

---

# Project Overview

EcoCycle was designed as a lightweight educational platform that encourages responsible waste management. The website aims to make recycling knowledge accessible and engaging through structured content and simple interactive features.

The platform provides:

- Educational information on recyclable materials
- Guidance on proper recycling preparation
- A feature to locate recycling centers
- An interactive quiz to test user knowledge

The interface emphasizes clarity, minimal design, and ease of navigation to ensure users can quickly access the information they need.

---

# Features

## 1. Recycling Guide
The recycling guide provides categorized information about different recyclable materials. Each material category expands to display instructions on how to recycle it properly.

Materials covered include:

- Plastic
- Wood
- Metal
- Organic (Food and Yard Waste)
- Glass

Users can click on each category to view recycling guidelines and preparation tips.

---

## 2. Recycling Center Locator
The website includes a section that helps users locate nearby recycling centers. A "Locate Now" button is provided which can be extended to integrate with mapping services such as Google Maps or browser geolocation APIs.

This feature is intended to make it easier for users to identify where they can properly dispose of recyclable materials. It is under development for now, kindly check back frequently to see updates 🙂‍↕️🎮

---

## 3. Interactive Recycling Quiz
EcoCycle includes a short quiz that tests users’ knowledge about recycling practices.

The quiz:

- Contains multiple-choice questions
- Evaluates user responses
- Displays the final score after submission
- Allows users to reset and retake the quiz

This feature encourages learning through engagement and helps reinforce recycling concepts.

# Technologies Used

The website was built using fundamental web technologies:

**HTML5**  
Used to structure the content and layout of the webpage.

**CSS3**  
Used for styling, layout design, and responsive formatting.

**JavaScript (Vanilla JS)**  
Used to implement interactivity including:
- Accordion functionality
- Quiz scoring logic
- Button actions
- Dynamic content display

No external frameworks or libraries were used.
